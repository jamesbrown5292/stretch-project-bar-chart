Welcome to James's Graph Generator!

About the graph generator

The graph generator is a simple app designed to demonstrate fundamental skills in html, css, javascript and jQuery. The generator takes in a series of data points from the user, and returns a graphical representation of the data, giving the user the ability to customize the graph's appearance.

How to use the graph generator

To get started with your graph, use the boxes below to create a data set, then watch as your graph appears.

Now, give your data series a title. This might be, for example, "types of fruit" or "dietary preferences"

Next, we need to enter some data. Use the input fields below to enter a category and a value for that category. For example, you might enter "apples, 13" or "pescatarians, 20"
